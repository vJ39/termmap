# termmap 操作マニュアル

OpenStreetMap のタイルを端末に描画する地図ツール。ルート・目的地(POI)・航続リング・お気に入りを組み合わせて、端末上でツーリング計画を立てられる。

## インストール / ビルド

```
cargo build --release
```

ビルドすると `target/release/termmap` にバイナリができる。以降はこのバイナリを実行する。

## 基本の使い方

住所や地名を指定して地図を出す。

```
termmap --place "東京都北区田端"
```

緯度経度で中心を指定することもできる。

```
termmap --lat 35.681 --lon 139.767
```

対話モードで起動すると、キー操作で地図を動かしながらルートを組み立てられる。

```
termmap --place "東京駅" -i
```

## 対話モードのキー一覧

### 移動
- `←` `↑` `↓` `→` パン(Shift+矢印で大きく移動)
- `+` `-` ズーム
- `/` 住所・地名で検索して移動
- `a` 中心の住所を表示

### ルート点
- `s` 始点
- `e` 終点
- `v` 経由点(終点手前に追加)

いずれも中心クロスヘアの位置に点を置く。

### 並べ替え
- `Tab` 並べ替えパネル(順序表示・選択は白丸で強調)
  - `↑` `↓` / `Tab` 選択
  - `[` `]` 前後へ移動
  - `x` 削除
  - `Esc` 閉じる

### ルート設定
- `m` モード(下道 → 高速 → 最短)
- `c` ルート消去
- `g` GPX保存(termmap-route.gpx)

### 目的地
- `f` カテゴリ検索(1ガソスタ 2カフェ 3コンビニ 4道の駅 5展望 6公園 7峠道)
  - リストで `↑` `↓` 選択
  - `s` 始点 / `e` 終点 / `Enter` 経由
  - `f` 再検索
  - `Esc` 閉じる

### お気に入り
- `S` ルート保存(名前入力)
- `L` 呼び出し一覧

### その他
- `?` ヘルプ
- `q` 終了
- `Esc` サブモード取消

## ツーリングの流れ(例)

1. `-i` で起動する。
2. `/` で出発地へ移動する。
3. `s` で始点を置く。
4. 目的地へ `/` で移動する。
5. `e` で終点を置く。
6. 経由したい所へ `/` で移動して `v` で経由点を置く。
7. `m` で下道 / 高速を切り替える。
8. `S` で名前を付けて保存する。
9. 次回は `--load-route 名前` で呼び出す。
10. スマホで走るときは `--load-route 名前 --share` で Googleマップ経路URL + QR を出し、スキャンしてマップアプリで開く(または `--gpx` で書き出して OsmAnd 等に読ませる)。

## 起動オプション(CLI)

- `--range KM,..` 航続リング
- `--home LAT,LON` リング基準
- `--route "LAT,LON;.."` ルート(始点;経由;終点)
- `--route-mode surface|highway|short` ルートのモード
- `--gpx OUT` GPX書き出し
- `--save-route N` 現在のルートを名前 N で保存
- `--load-route N` お気に入り N を読み込む
- `--routes` お気に入り一覧
- `--share` ルートを Googleマップ経路URL + 端末QR で出力(スマホでスキャンして開く)
- `--here` 現在地を中心にする
- `--resume` 前回終了時の状態を復元
- `--style osm|voyager|dark|light` タイル種別
- `--edge` 輪郭抽出
- `--classify` 地物カテゴリ色分け
- `--braille` 点字ドット描画
- `--width N` 出力桁数

## データと通信

- 地図タイル: OpenStreetMap
- ルート: BRouter
- 目的地(POI): Overpass
- 住所検索 / 逆ジオコーディング: Nominatim

いずれもオンラインAPIを使う。タイルはセッション内のメモリキャッシュのみで、再起動すると最新のOSMを取り直す。オフラインには対応していない。

## 注意

- 高速料金は概算。高速区間の距離 × ¥30 で、普通車・割引なしの前提で計算する。実額とは異なる。
- 下道モードは BRouter の moped プロファイルを使い、高速を回避する。
