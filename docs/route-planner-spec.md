# termmap 対話ルートプランナー 仕様（ドラフト）

-i 対話モード内でルートを組み立て・編集するUI。コマンド引数を打ち直さず、机上でツーリング計画する。

## 状態

- `waypoints: Vec<Waypoint>` … 順序付き。始点=先頭, 終点=末尾, 間=経由点
  - `Waypoint { lat, lon, label }`（label=住所/POI名, 無名可）
- `mode` … short(shortest) / quiet(car-eco) / 任意プロファイル名。自動車前提
- `route` … 計算結果（点列 + 距離m + 時間s）キャッシュ
- `pois` … 目的地検索の結果リスト（カテゴリ別）
- `sel` … 選択中 waypoint の index（Tab で巡回、編集/削除の対象）
- 中心クロスヘア常時表示（s/e/w をどこに落とすか可視化）

## キーバインド（案）

既存: `←↑↓→`pan / `Shift`大 / `+ -`zoom / `/`住所検索移動 / `a`中心住所 / `q`終了

ルート編集:
- `s` … 現在中心を **始点** に（先頭を置換）
- `e` … 現在中心を **終点** に（末尾を置換）
- `w` … 現在中心を **経由点** として終点の手前に追加
- `Tab` … waypoint 選択を巡回（選択点を白枠ハイライト）
- `x` … 選択中 waypoint を削除
- `i` … 選択点の **手前に割り込み挿入**（現在中心を経由点として）
- `m` … モード切替 short/quiet（ルートあれば再計算）
- `c` … ルート全消去
- `g` … GPX 書き出し

目的地検索(POI):
- `f` … カテゴリ選択（1..7）→ 現在表示範囲で Overpass 検索 → リスト表示
- リスト内: `↑↓`選択 / `Enter`経由点として追加 / `E`終点に設定
- POIマーカーを地図に重畳（カテゴリ色）

航続リング:
- `R` … 始点（無ければ中心）基準でリング表示/非表示（--range 値 or 既定 例:20,40km）

## 目的地検索（POI系）詳細

- カテゴリ（充電は除外確定）: 1=ガソスタ 2=カフェ 3=コンビニ 4=道の駅 5=展望 6=公園 7=給水
- 取得: 現在表示 bbox で Overpass（UA/Accept 必須, `out center;`）。道の駅は name で重複排除
- 表示: 下部に数行 or 左袖に縦リスト。各行「名前 / 種別 / 中心からの距離km」、距離順ソート
- 選択して経由/終点へ。追加後リストは残す（連続追加可）

## ルート計算

- `waypoints >= 2` で自動計算。点の 追加/削除/移動/割り込み/モード変更 のたび再計算
- BRouter: `lonlats=` 全 wp を **lon,lat 順**で `|` 連結, `profile=mode`, `format=geojson`
- 距離/時間をステータス表示。将来: 歩道(footway)混入区間の警告
- ネットワーク失敗時は地図は保持しルートのみスキップ

## 描画

- ルート=水色線 / 始点=緑 / 終点=赤 / 経由=橙 / 選択点=白枠 / 中心=黄クロスヘア
- POI=カテゴリ色マーカー / 航続リング=赤

## 「割り込み」の意味

既存ルートの途中に経由点を挿入して迂回させる。当面は `i`＝選択点の手前に現在中心を挿入。
将来: 地図上の最寄りセグメントへ自動挿入。
